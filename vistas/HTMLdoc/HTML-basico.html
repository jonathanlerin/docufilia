¿Qué es HTML?

HTML es un lenguaje de programación utilizado para el desarrollo web. Su principal función es la maquetación de páginas web siendo el esqueleto de cualquier web.

Aquí encontrarás un lugar de referencia para hacer consultas rápidas sobre html.

¿Cómo usar un elemento HTML?

Un elemento html consiste por una etiqueta de apertura y una etiqueta de cierre. Entre estás etiquetas se encuentra el contenido. Las etiquetas definen o especifican al contenido que contienen.

La sintaxis sería <etiqueta> contenido </etiqueta>. Por ejemplo:

<h2> Esto es un título de nivel dos siendo el segundo más grande</h2>.

Analicemos este caso:

  • <h2></h2> Define qué es. En este caso un título de nivel dos.
  • "Esto es un titulo de nivel dos siendo el segundo más grande". Es el contenido y mientras no este definido por unas etiquetas de un elemento HTML podría ser cualquier cosa pero en cuanto se encuentra entre las etiquetas de apertura y cierre de h2 pasa a ser para el navegador un título de nivel dos.

NOTA: Lo repetiré muchas veces a lo largo de esta web pero está es la primera. HTML es un lenguaje del lado del cliente. Es decir el código se ejecuta en nuestro navegador.

Enlaces

Un enlace es un texto o elemento que al pulsar sobre él nos lleva a otra página web o a otra zona de la misma página. Cuando el enlace nos lleve a otra página lo llamaremos enlace externo, cuando nos enlace dentro de la misma página ya cargada pero a otra zona lo denominaremos enlace interno.

Para enlazar un texto o elemento usaremos la etiqueta <a>.

Imágenes

Para poder usar imágenes dentro de nuestro documento HTML deberemos usar la etiqueta <img>. Debemos complementar esta etiqueta con la propiedad src="ruta_de_la_imagen" para indicar en donde se ubica la imagen.

Tablas

HTML también cuenta con etiquetas para poder definir un contenido formateado dentro de una tabla. Para este propósito utilizaremos las siguientes:

  • <table>: define una tabla.
  • <tr>: define una fila.
  • <th>: define una columna. Se utiliza en la primera fila y corresponde al título de la columna.
  • <td>: define una columna genérica.

Listas

  • <ul>: define una lista sin orden.
  • <ol>: define una lista ordenada.
  • <li>: define el elemento de una lista.

Contenedor neutro o bloque

Cuando empezamos a programar un documento HTML, a penas llevemos unas pocas lineas nos veremos en la necesidad de agrupar elementos para poder manejarlos de forma conjunta o en bloque.

Para ello utilizaremos la etiqueta <div> cuya función sirve para indicar al navegador que todos los elementos que contenga están relacionados y deberán ser tratados como un bloque. Por tanto sirve como un contenedor genérico que no tiene efectos colaterales.

Los <div> por defecto se muestras apilados unos encima de otros de manera que no se superponen.

Los <div> pueden contener otros <div>.
